## 🔧 How basher works

basher listens to your commands typed in plain language. It then converts these to bash commands you would normally run yourself in a terminal. The app runs these commands in Git Bash and returns results in real time.

Examples:

- You say: "Show me my network settings"
- basher runs: `ipconfig`
- You see the results right away

This approach makes simple automation accessible without knowing command-line programming.
